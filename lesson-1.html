5. *Самостоятельно разобраться с атрибутами тега script (async и defer).

1. Пример подключения async:
<script async src="js/main.js"></script>

С атрибутом async содержимое страницы HTML загружается одновременно с выполнением js-скрипта, подключённого из внешнего файла. Указанный в атрибуте src файл будет выполняться без ожидания загрузки и отображения веб-страницы. И поэтому загрузку HTML нет необходимости останавливать, когда парсинг достигает тега <script>. Работает этот атрибут только для внешне подключённого js-файла.

2. Пример подключения defer:
<script defer src="js/main.js"></script>

Атрибут defer откладывает выполнение js-скрипта до тех пор, пока не будет загружена полностью вся страница HTML. И даже если файл скрипта будет полностью загружен ещё до того, как парсер закончит работу, он не будет выполнен до тех пор, пока парсер не откроет страницу HTML до конца.
В браузере Firefox 3.6 и старше атрибут defer игнорируется, если у тега <script> не указан атрибут src. В браузере Firefox 3.5 defer работает даже для встроенных скриптов.
